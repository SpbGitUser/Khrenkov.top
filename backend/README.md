# Khrenkov.top Backend

Backend-часть проекта находится в `Khrenkov.top` и содержит ASP.NET Core 8 API для авторизации, загрузки, удаления, скачивания файлов и публичных share-ссылок.

## Запуск

```powershell
dotnet run --project Khrenkov.top/Khrenkov.top.csproj
```

Если запускать из корня репозитория:

```powershell
dotnet run --project backend/Khrenkov.top/Khrenkov.top.csproj
```

По умолчанию backend поднимается на `http://localhost:5214`.
